## Executive Summary

The Bumble Profile and User Behaviour Analysis project was undertaken to explore user-generated profile data and uncover meaningful insights into demographics, lifestyle preferences, profile completeness, and engagement patterns. The dataset contained missing values, inconsistent data types, invalid income records, and outliers, making data preprocessing a critical first step before conducting reliable analysis.

The dataset was cleaned by handling missing values, correcting data types, replacing invalid income values, treating outliers, and engineering new features such as age groups, income categories, profile completeness, and height in centimeters. Exploratory Data Analysis (EDA) was then performed using Python libraries including Pandas, NumPy, Matplotlib, and Seaborn to analyze demographic distributions, lifestyle habits, geographical trends, and relationships between variables. A wide range of statistical summaries and visualizations were created to communicate findings effectively.

The analysis revealed that users aged **26–35** form the largest segment of the platform, while over **90%** of profiles belong to users who identify as single, aligning with Bumble's primary target audience. A moderate positive relationship between age and income suggests that reported income generally increases with age. Lifestyle analysis showed that **social drinking** and **"mostly anything" diets** are the most common preferences, while California, particularly San Francisco, has the highest user concentration. Additionally, profile completeness was found to be high for most users, indicating strong user participation in profile creation. These insights can support improved user segmentation, personalized recommendations, targeted marketing strategies, and data-driven product enhancements.
