## Executive Summary

- Analyzed Bumble user profile data to understand user demographics, lifestyle preferences, profile quality, and behavioral patterns that can support product growth, user engagement, and personalization strategies.
- Performed end-to-end data preprocessing by handling missing values, invalid income records, inconsistent data types, duplicates, and outliers to improve data quality and analytical reliability.
- Engineered business-focused features including **Age Groups, Income Categories, Profile Completeness Score,** and **Height (cm)** to enable meaningful user segmentation and behavioral analysis.
- Conducted Exploratory Data Analysis (EDA) using **Python, Pandas, NumPy, Matplotlib, and Seaborn**, analyzing demographic distribution, geographic concentration, lifestyle habits, income trends, and variable relationships.
- Identified **26–35 years** as the platform's largest user segment, highlighting the primary demographic for acquisition and engagement initiatives.
- Found that **90%+** of users identify as **Single**, validating Bumble's core target audience and supporting focused product positioning.
- Discovered a **moderate positive correlation between age and income**, indicating higher earning potential with increasing age and enabling income-based user segmentation.
- Identified **Social Drinking** and **Mostly Anything** diet preference as the most common lifestyle attributes, providing valuable inputs for recommendation engines and profile matching algorithms.
- Observed **California**, particularly **San Francisco**, as the region with the highest user concentration, indicating strong geographic adoption and opportunities for location-specific marketing campaigns.
- Found that the majority of users maintain **high profile completeness**, suggesting strong onboarding engagement and richer user profiles for recommendation and matching models.
- Delivered actionable insights that can improve **user segmentation, personalized recommendations, targeted marketing campaigns, user acquisition strategy, and future product enhancements** through data-driven decision making.
