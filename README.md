## Executive Summary

- Analyzed Bumble user profile data to uncover demographic patterns, lifestyle preferences, income distribution, and profile completeness, enabling data-driven user segmentation and product enhancement strategies.
- Cleaned and preprocessed the dataset by handling missing values, correcting data types, replacing invalid income values, treating outliers, and engineering new features to improve analytical accuracy.
- Applied **Python, Pandas, NumPy, Matplotlib,** and **Seaborn** to perform exploratory data analysis, descriptive statistics, correlation analysis, feature engineering, and create **30+ analytical visualizations** for business storytelling.
- Evaluated key metrics including **Age Distribution, Gender Ratio, Relationship Status, Income Distribution, Profile Completeness, Lifestyle Preferences, Height Analysis,** and **Geographic User Distribution** to understand platform behavior.
- Identified users aged **26–35** as the largest demographic segment, while **male users accounted for approximately 60%** of the platform and **over 92% of users identified as single**, validating Bumble's core target audience.
- Found **Social Drinking** and **"Mostly Anything"** to be the most common lifestyle preferences, providing valuable insights for personalization and recommendation algorithms.
- Observed a **moderate positive correlation (r = 0.38)** between **age and income**, indicating that reported income generally increases with age and supporting income-based user segmentation.
- Discovered that **California**, particularly **San Francisco**, had the highest user concentration, highlighting key geographic markets for targeted user acquisition campaigns.
- Measured an average **profile completeness of approximately 85%**, indicating strong user participation and high-quality profile data for matchmaking and recommendation models.
