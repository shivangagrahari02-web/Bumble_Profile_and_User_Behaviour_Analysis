# Bumble Profile and User Behaviour Analysis

A Python-based exploratory data analysis project on Bumble user profiles to uncover demographic patterns, lifestyle preferences, income trends, and profile completeness insights.

## Overview
This project analyzes Bumble profile data to understand user behavior, preferences, and platform trends. The dataset includes demographic, lifestyle, financial, and activity-related fields such as age, gender, status, height, diet, drinks, income, location, and last online activity.

The goal is to clean the data, create useful derived features, explore relationships between variables, and present findings through meaningful visualizations. The analysis helps identify user segments, missing data patterns, and key behavioral trends that can support better matchmaking and product decisions.

## Problem Statement
Bumble wants to better understand its users by analyzing profile data and uncovering meaningful patterns in demographics, lifestyle choices, income, and activity behavior.

This project aims to answer questions such as:
- Which profile fields have the most missing values?
- What is the age, gender, and relationship status distribution?
- How do income, height, and lifestyle preferences vary across users?
- What patterns appear in location, pet preferences, and zodiac signs?
- How complete are user profiles overall?

## Dataset
The dataset contains Bumble user profile information with the following major fields:
- Demographics: age, gender, status
- Physical attributes: body_type, height
- Lifestyle preferences: diet, drinks, smoking, pets
- Educational and financial details: education, income
- Location and activity: location, last_online
- Additional profile fields used for behavioral analysis

## Key Insights

- Cleaned and preprocessed the Bumble dataset by handling missing values, correcting data types, replacing invalid income values, and treating numerical outliers.
- Users aged **26–35** represent the largest demographic, accounting for nearly half of the platform's user base.
- **Male users (60%)** slightly outnumber female users (40%), while over **92% of users identify as single**, confirming Bumble's primary audience.
- **Social drinking** is the dominant lifestyle preference, regardless of dietary habits, while **"Mostly Anything"** is the most common diet choice.
- Age and income show a **moderate positive correlation (0.38)**, indicating that reported income generally increases with age.
- California has the highest concentration of users, with **San Francisco** being the most active city in the dataset.
- Profile completeness averages around **85%**, indicating that most users provide sufficient information for meaningful analysis.
- Median income increases consistently across age groups, while male users report slightly higher median incomes than female users.
- Height remains relatively stable across age groups but varies significantly between genders.

## Results & Conclusion

This analysis transformed raw Bumble profile data into meaningful business insights through data cleaning, feature engineering, exploratory analysis, and visualization.

The findings reveal that Bumble's user base is primarily composed of **single users aged 26–35**, with California contributing the highest number of profiles. Lifestyle analysis shows that social drinking is common across most dietary groups, while financial analysis indicates that reported income generally increases with age.

The project also demonstrates the importance of handling missing values, invalid entries, and outliers before performing statistical analysis. Overall, the notebook provides a comprehensive understanding of user demographics, preferences, and behavioural trends that can support better matchmaking, user segmentation, and product decision-making.
